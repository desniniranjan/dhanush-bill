# Dhanush Bill Manager

A smart billing and invoice management application built with React, TypeScript, and Supabase.

## Features

- 📄 Create and manage bills/invoices
- ⚡ Fast and responsive UI built with Vite + React
- 🔐 Secure authentication with JWT and Supabase
- 🗄️ Real-time database powered by Supabase

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Backend:** Node.js, Express
- **Database:** Supabase
- **Auth:** JWT
- **Styling:** CSS

## Getting Started

### Prerequisites

- Node.js (v18 or above)
- A [Supabase](https://supabase.com) project
